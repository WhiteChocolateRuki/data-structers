*** Student Management System Using File-Based Singly Linked List ***


This project is a student course management system implemented in the C programming language.  
It uses **`struct`-based singly linked lists** to store and manipulate student records and the courses they are enrolled in.  
It loads the data from a plain text file (`metin.txt`) and offers a simple menu-based console interface for operations.



Features of the Automation Project:
The user can:
-Load student and course data from a file
-Add new student and course information
-Delete students by surname
-List students sorted by their student number
-List students enrolled in a specific course
-Search for a student by name
-Display midterm and final exam grades


***Dosya Tabanlı Tek Yönlü Bağlantılı Liste ile Öğrenci Yönetim Sistemi ***
Bu proje, C programlama diliyle yazılmış bir öğrenci-ders yönetim sistemidir.  
Öğrenci ve ders verileri **tek yönlü bağlı liste (singly linked list)** yapısı ve `struct` kullanılarak modellenmiştir.  
`metin.txt` adlı bir düz metin dosyasından veri çekilir ve konsol menüsü üzerinden yönetim yapılır.



### Otomasyon Projesinin Özellikleri:
Kullanıcı
-Dosyadan öğrenci ve ders verilerini yükleme
-Yeni öğrenci ve ders bilgisi ekleme
-Soyada göre öğrenci silme
-Öğrencileri numaraya göre sıralı listeleme
-Belirli bir dersi alan öğrencileri listeleme
-İsimle öğrenci arama
-Vize ve final notlarını gösterme 
işlemlerini yapabilmektedir.
