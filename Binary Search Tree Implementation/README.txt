***Binary Search Tree Implementation with File Input *** 


This project implements a **Binary Search Tree (BST)** where the initial node is read from a file called `sayi.txt`.  
The user can perform standard operations like inserting values, deleting nodes, traversing the tree in-order, and tracking the path taken to find a specific value.

This project demonstrates how BSTs work and how file-based initialization can integrate with tree structures.

### Data Structure Info:
- **Binary Search Tree (BST)**  
- Each node has left, right, and parent pointers  
- Recursive insert, delete, and search operations  
- No stack, queue, or balancing (like AVL or Red-Black Tree)

###Features:
- Reads the root value from `sayi.txt` (e.g., `42`)
- Add new integers to the BST
- Delete existing values from the tree
- In-order traversal of the entire tree
- Trace and print the search path to a given value
- Console menu for user interaction

***Dosyadan Başlatmalı İkili Arama Ağacı Uygulaması***

Bu proje, **ikili arama ağacı (Binary Search Tree)** yapısını uygular ve başlangıç değeri `sayi.txt` adlı dosyadan alınır.  
Kullanıcı, yeni sayılar ekleyebilir, silebilir, ağacı sıralı biçimde görüntüleyebilir veya belirli bir değere ulaşana kadar izlenen yolu görebilir.

Bu proje, ikili arama ağacının nasıl çalıştığını ve dosya üzerinden nasıl başlatılabileceğini gösteren güzel bir örnektir.

### Veri Yapısı Bilgisi:
- **İkili Arama Ağacı (BST)**  
- Her düğümde sol, sağ ve ebeveyn göstericileri bulunur  
- Ekleme, silme ve arama işlemleri özyinelemeli yapılır  


### Özellikler:
- Başlangıç değeri `sayi.txt` dosyasından alınır (örn: `42`)
- Kullanıcı ağaç içine yeni değerler ekleyebilir
- Var olan düğümleri silebilir
- Ağacı sıralı şekilde dolaşabilir (in-order traversal)
- Belirli bir değere ulaşırken izlenen yolu ekrana yazdırabilir
- Kullanıcı dostu konsol menüsü ile çalışır
